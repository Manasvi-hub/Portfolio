# 🚀 Manasvi Yeole Portfolio

A futuristic personal portfolio website showcasing my journey as a Software Engineer, Web Developer, and AI/ML Enthusiast. Built with a modern cyber-inspired design, interactive UI elements, glassmorphism effects, dynamic animations, and responsive layouts.

## 🌟 Features

* Modern futuristic UI inspired by AI dashboards
* Responsive design for desktop and mobile devices
* Interactive project showcase
* Animated particle cursor effects
* Glassmorphism cards and neon accents
* Skills dashboard
* About Me section
* Contact section with social links
* Resume access
* Smooth navigation between sections

## 🛠️ Technologies Used

### Frontend

* HTML5
* CSS3
* JavaScript (Vanilla)

### Design & UI

* Orbitron Font
